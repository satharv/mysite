Ever wondered where is the information of user is saved when we create a user in Linux. Today we will take a look at 4 files which contains user information when a user is created.
All this files are located in etc folder.
1. /etc/passwd
2. /etc/group
3. /etc/shadow
4. /etc/gshadow
